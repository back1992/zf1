Introduction

You can use PHP_Beautifier in many ways:

  • Inside a script, using a instance of the class PHP_Beautifier or a subclass of PHP_Beautifier_Decorator, like PHP_Beautifier_Batch. See PHP_Beautifier.howtouse.script.pkg

  • As a command line tool. By default, the installer puts the script in the path of executables, so you can call it directly using 'php_beautifier'. See PHP_Beautifier.howtouse.commandline.pkg
