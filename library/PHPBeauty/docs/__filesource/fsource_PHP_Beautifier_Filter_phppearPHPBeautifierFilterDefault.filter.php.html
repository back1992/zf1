  26.     /**
    27. 

  27.     * Default Filter
    28. 

  28.     * This filters is loaded by default in {@link PHP_Beautifier}. Can handle all the tokens.
    29. 

  29.     * If one of the tokens doesn't have a function, is added wihout modification (See {@link __call()})
    30. 

  30.     * The most important modifications are:
    31. 

  31.     * - All the statements inside control structures are indented
    32. 

  32.     * - The function and class defitions have this structure
    33. 

  33.     * <CODE>
    34. 

  34.     * function myFunction() {
    35. 

  35.     *     echo 'hi';
    36. 

  36.     * }
    37. 

  37.     * </CODE>
    38. 

  38.     * - All the comments in new lines are indented. In multi-line comments, all the lines are indented, too.
    39. 

  39.     * This class is final, so don't try to extend it!
    40. 

  40.     * @package PHP_Beautifier
    41. 

  41.     * @subpackage Filter
    42. 

  42.     */
