  27.     /**
    28. 

  28.     * Create a list of functions and classes in the script
    29. 

  29.     * By default, this Filter puts the list at the beggining of the script.
    30. 

  30.     * If you want it in another position, put a comment like that

  31.     * <pre>

  32.     * // Class and Function List

  33.     * </pre>

  34.     * The script lookup for the string 'Class and Function List' in a comment and replace the entire comment with the list
    35. 

  35.     * The settings are
    36. 

  36.     * - list_functions: List functions (0 or 1). Default:1
    37. 

  37.     * - list_classes:   List classes (0 or 1).   Default:1
